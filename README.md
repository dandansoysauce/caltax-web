# CALTAX
PH Withholding Tax Calculator with salary deductions (**SSS, Philhealth and Pag-Ibig**) under TRAIN Law.

## Building
```
$ git clone https://github.com/dandansoysauce/caltax-web.git
$ cd caltax-web
$ yarn install
$ yarn start
```
